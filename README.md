# Amazon Reviews Sentiment Analysis

## Project Overview
This project performs sentiment analysis on Amazon customer reviews to understand customer opinions and feedback. Reviews are classified into Positive, Neutral, and Negative sentiments based on customer ratings, followed by visual analysis to uncover trends and patterns.

## Objectives
- Analyze customer reviews and ratings.
- Classify sentiments as Positive, Neutral, or Negative.
- Explore sentiment distribution across the dataset.
- Identify common words used in positive and negative reviews.
- Generate insights through data visualization.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- WordCloud
- Google Colab

## Dataset Features
The dataset contains:
- Reviewer Name
- Country
- Rating
- Review Title
- Review Text
- Review Date
- Date of Experience

## Analysis Performed
- Data Cleaning and Preprocessing
- Rating-Based Sentiment Classification
- Sentiment Distribution Analysis
- Rating Distribution Analysis
- Country-wise Review Analysis
- Review Length Analysis
- Positive Review Word Cloud
- Negative Review Word Cloud

## Visualizations
- Sentiment Distribution Bar Chart
- Sentiment Share Pie Chart
- Rating Distribution Chart
- Top Countries by Reviews
- Review Length Distribution
- Positive Word Cloud
- Negative Word Cloud

## Key Insights
- Most reviews were classified as positive.
- Higher ratings dominate the dataset.
- Customer feedback is generally favorable.
- Word clouds reveal frequently mentioned positive and negative terms.
- Review patterns help understand customer satisfaction trends.

## Outcome
This project demonstrates the application of Natural Language Processing (NLP) and data visualization techniques to analyze customer feedback and extract actionable insights from review data.
